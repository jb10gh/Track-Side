# Coding Conventions

- **Mobile First:** All components must be designed for 375px targets.
- **Large Touch:** Minimum 44px height for interactive elements.
- **State Mutators:** Prefer granular Zustand actions over large state overwrites.
- **Component Structure:** `src/components/{area}/{Name}.jsx`
- **Tailwind:** Use `@theme` for custom brand colors in v4.
- **Aesthetics:** High contrast OLED black backgrounds.
