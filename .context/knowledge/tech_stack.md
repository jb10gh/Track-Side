# Tech Stack: Sideline Stats V3

- **Framework:** React 19 + Vite
- **Styling:** Tailwind CSS v4 + Vanilla CSS (for theme-specific glow/glass utilities)
- **State:** Zustand 5+ (Persistent)
- **Icons:** Lucide React
- **Routing:** React Router (HashRouter for GH Pages compatibility)
- **Design:** OLED Dark Mode (#000000) with Glassmorphism
- **PWA:** `vite-plugin-pwa`
